Revolut Budgetting App

AUG 2022

App logo created using DALL·E 2

Coded a MATLAB app that lets you see statistics of your Revolut statement (downloaded from their website as a .csv file). The app lets you filter the data and also create text to input in https://sankeymatic.com/build/ to get a Sankey diagram of your statement.

The data can be grouped so that the Sankey diagrams are not so filled with nodes that area impossible to read.

You can find the app here.

Voronoi Tessellation

MAR 2022

I developed a code in MATLAB that creates Voronoi Tessellations in 2D and 3D. The 2D ones can be easily visualised in MATLAB plots but the 3D ones are difficult, so I used ParaView to visualise them.

The code just creates random centres inside the grid (the red dots in the 2D Voronoi Tesellation) and iteratively goes pixel by pixel and finds the centre that is the closest and paints it the same color.

Go to this GitHub repo to download the code.

Crystal Viewer

JUN 2021

Did a MATLAB app that can be used to visualise and create unit cells from crystals. The UI is very bare-bones and there are not that many functionalities except viewing miller directions and planes.

The code is supposed to work with all crystal systems, but the only ones I am sure it works with is cubic, tetragonal and orthorombic systems.

It is difficult to understand anything when there are a lot of atoms, and I did not add the capability to change the size of the atomic species which would greatly improve visibility in some cases.

The program can be installed from here (only available for Windows, unless you have MATLAB then you can download the source code from the repository and run it anywhere).
