Voronoi Tessellation

MAR 2022

I developed a code in MATLAB that creates Voronoi Tessellations in 2D and 3D. The 2D ones can be easily visualised in MATLAB plots but the 3D ones are difficult, so I used ParaView to visualise them.

Go to this GitHub repo to download the code.
